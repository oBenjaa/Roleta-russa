### Roleta Russa

**O Roleta Russa é um jogo de terminal desenvolvido em TypeScript, inspirado no clássico jogo de tensão e sorte. O jogador participa de rodadas onde deve decidir se deseja continuar e puxar o gatilho ou abandonar o jogo. A cada rodada, o tambor do revólver avança e o resultado pode ser apenas um: vida ou Derrota.**

### **O projeto foi criado com foco em:**

**Lógica de programação**

**Manipulação de entrada de usuário**

**Uso de classes e modularização no TypeScript**

**Exercitar o uso de Node.js em aplicações de console**

### Como jogar:

- Tenha o [Node](https://nodejs.org/pt) instalado em sua máquina
- Abra um novo projeto em um editor de código de sua preferência.
- Clone o repositório usando o seguinte comando:
- ``git clone git@github.com:oBenjaa/Roleta-russa.git``
